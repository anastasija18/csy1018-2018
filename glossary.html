Web development terms glossary

HTML

HTML stands for HyperText Markup Language.

CSS

CSS stands for Cascading Style Sheet.

JavaScript

JavaScript is a sripting lanuague that allows designers to create interactions on a web page.

AJAX

AJAX stands for Asynachronous JavaScrip and XML.

Anchor text

The text a link uses to refer to a site.

Back end

The back end of a website is the part hidden from view of regular website visitors.

Client-side

The client-side refers to scripts that run in a viewer's browser, instead on a web server.

Comment

A comment is a bit of information contained in a site's HTML files.

Doctype

The doctype declartion specifies which version of HTML is used in the document.

Domain

The domain is the name by which a website is identified.

E-Commerce

Is short for electronic commerce.

Element

An element is the central building block of any document. Individual elements can contain text, other elements or both.

Font family

Font family is a group designation for defining the typefaces useed in CSS documents.

Font style

In CSS, the font style refers to whether a font is italic or not.

Font weight

The font weight refer to how thick or thin a font looks.

Front end

The front end is all the components of a website that a visitor to the website can see.

HTTP

Stands for HyperText Transfer Protocol.
